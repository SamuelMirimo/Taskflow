## TaskFlow - Gestionnaire de tâches Full-Stack

Application web de gestion de tâches avec backend Express.js/MySQL et frontend html, css et JavaScript.

## Fonctionnalités

Backend (API REST)
- CRUD complet des tâches (Create, Read, Update, Delete)
- Connexion MySQL avec pool de connexions
- Architecture MVC propre
- Gestion d'erreurs

Frontend
- Interface responsive et intuitive
- Modal pour création
- Toggle statut terminé/en cours
- Suppression avec confirmation

## Technologies

Backend :
- Node.js
- Express.js
- MySQL
- mysql2

Frontend :
- HTML5
- CSS3
- JavaScript

## Installation

1. Prérequis
- Node.js
- MySQL (WAMP ou un autre)
- Git

2. Cloner le projet
git clone https://github.com/TON_USERNAME/taskflow.git
cd taskflow

3. Configuration
cd bakend
npm install

NB : vous trouverez la structure de la BD dans le fichier backend/database/init.sql

## lancer le serveur 
npm run dev
